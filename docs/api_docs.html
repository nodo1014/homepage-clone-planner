GET /export/{result_id}/{format_type}

요약: Export Results

분석 결과를 지정한 형식으로 내보내기

Args: result_id (str): 결과 ID format_type (str): 내보내기 형식 (markdown, html, pdf, pptx, json, zip, notion, gdrive)

Returns: 각 형식에 맞는 응답: - notion: JSONResponse (노션 페이지 URL 반환) - gdrive: JSONResponse (구글 드라이브 URL 반환) - 그 외: RedirectResponse (다운로드 URL로 리다이렉트)

매개변수

| 이름 | 위치 | 타입 | 필수 | 설명 |

|------|------|------|------|------|

| result_id | path | string | ✓ | |

| format_type | path | string | ✓ | |

응답

200 - Successful Response
  • 미디어 타입: application/json
422 - Validation Error

  • 미디어 타입: application/json

  • 스키마:

{
  "$ref": "#/components/schemas/HTTPValidationError"
